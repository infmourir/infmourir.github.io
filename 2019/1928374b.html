2019.8 华师大二附中2019学年信息学夏令营 数论部分练习题

发表于 2019-08-03 | 分类于 OI | 评论数：
去年盛开的海棠竟还未发芽，仿佛倒退了那张证书天真的受伤。夕露褶皱谁的脸庞，笨拙的我还不敢想，只够欣赏，黑夜中的微光~

包括讲义及习题讲解视频

Part I

1. BZOJ 2818 Gcd

给定整数 $n$，求 $1 \le x, y \le n$ 且满足 $gcd(x, y)$ 为 素数 的数对 $(x, y)$ 有多少对。

$n \le 10^7$

2. POJ 2480 Longge’s Problem

给定整数 $n$，求 $\sum^{n}_{i=1}{gcd(i, n)}$ 的值。

$ 1 < n < 2^{31}$

3. POJ1061 青蛙的约会

求使得同余方程 $x + mt \equiv y + nt(mod \ L)$ 成立的最小正整数 $t$。

$x, y, m, n, L$ 均在 $32$ 位整数范围内。

4. CH3B04 Xiao 9*大战朱最学

解一个方程数不超过 $10$ 的线性同余方程组 $x \equiv b_i (mod \ a_i)$。

$a_i, b_i \le 1200000$，$a_i$ 两两互质。

5. BZOJ2242 计算器

要求设计一个计算器，完成以下三项任务：

  1. 给定 $y, z, p$，计算 $y^z \ mod \ p$ 的值；
  2. 给定 $y, z, p$，计算满足 $xy \equiv z(mod \ p)$ 的最小非负整数 $x$；
  3. 给定 $y, z, p$，计算满足 $y^x \equiv z(mod \ p)$ 的最小非负整数 $x$。

其中 $y, z, p \le 10^9$ 且为质数。

Part II

6. POJ3233 Matrix Power Series

给定一个 $n \times n$ 大小的矩阵 $A$ 和正整数 $k$，求 $S = A + A^2 + … + A^k$。

$n \le 30, k \le 10^9$

7. HDOJ5015 233 Matrix

一个 $n + 1$ 行 $m + 1$ 列的矩阵 $A_{0, 0 \rightarrow n, m}$，其中 $A_{0, 1} = 233, A_{0, 2} = 2333, …$ ，$A_{1, 0}, A_{2, 0}, … , A_{n, 0}$ 由输入决定，其余元素满足 $A_{i, j} = A_{i, j - 1} + A_{i - 1, j}$。试计算 $A_{n, m}$。

$n \le 10, m \le 10^9$

8. POJ2947 Widget Factory

给定 $n$ 个未知数 $x_1, x_2, …, x_n \in \{ 3, 4, 5, 6, 7, 8, 9 \}$ 和 $m$ 个模 $7$ 意义下的模线性方程，试求解该方程组。

$n, m \le 300$

9. BZOJ2115 XOR

给定一个边权为非负整数的无向连通图，试求一条从结点 $1$ 到结点 $n$ 的路径（可以重复经过结点），使得路径上经过的边权值的异或和最大。

$n \le 50000, m \le 100000$

10. BZOJ3105 新NIM游戏

定义新 $NIM$ 游戏：第一个回合中，先手可以直接拿走若干整堆火柴（可以不拿，不可以全部拿走），第二回合后手也可以做一次同样的操作。从第三个回合开始和传统 $NIM$ 游戏规则相同。问先手是否有必胜策略，以及保证必胜的前提下第一回合拿走火柴总数的最小可能值。

火柴堆数 $k \le 100$。

Part III

11. BZOJ4517 排列计数

求所有长度为 $n$ 且恰好存在 $m$ 个 $a[i] = i$ 的排列的个数。

$n, m \le 10^6$，$500000$ 组数据。

12. POJ3904 Sky Code

给定 $n$ 个正整数，求其中满足 $(a, b, c, d) = 1$ 的四元组 $(a, b, c, d)$ 个数。

$n \le 10000$

13. CH3B13 守卫者的挑战

初始状态下你有 $k$ 的贡献。有 $n$ 个挑战，每个挑战中你有 $p_i$ 的概率成功并得到 $a_i$ 的贡献，求最终贡献和 $S \ge 0$ 且成功的挑战数不小于 $L$ 的概率。

$0 \le k \le 2000, 0 \le n \le 200, -1 \le a_i \le 1000, 0 \le L \le n$

14. CH3B14 换教室

有 $2n$ 节课程安排在 $n$ 个时间段上。在第 $i$ 个时间段上，两节内容相同的课程同时在不同的地点进行，其中，牛牛预先被安排在教室 $c_i$ 上课，而另一节课程在教室 $d_i$ 进行。至多可申请更换 $m$ 节课程，且第 $i$ 个时间段上申请通过概率为 $k_i$。共有 $v$ 个教室，$e$ 条带权道路，每条双向道路连接两间教室。由于道路的长度和拥堵程度不同，通过不同的道路耗费的体力可能会有所不同。 试求申请哪几门课程可以使在教室间移动消耗权值总和期望值最小。

$n, m \le 2000, v \le 300, e \le 90000$

15. POJ2976 Dropping tests

有两个长为 $n$ 的数组 $a_i, b_i$，选择至少 $n - k$ 个下标 $d_1, …, d_m$，使得$f = 100 \cdot \frac{ \sum_{i = 1}^{m}a_{d_i} }{\sum_{i = 1}^{m}b_{d_i}}$最大。

$k \le n \le 1000$

16. CH3B16 魔法珠

有 $n$ 堆石子，分别有 $a_i$ 颗，两人轮流选择一堆颗数 $x > 1$ 的石子并拿走，同时加入 $d(x) - 1$ 堆颗数互不相同的石子，每一堆石子的颗数分别为拿走石子数的一个真约数。无法操作者输。问谁有必胜策略。

$n \le 100, a_i \le 1000$

17. POJ1704 Georgia and Bob

有若干个格子排成一排，其中从左到右第 $a_i$ 个格子放有棋子（共 $n$ 个）。两人轮流选择一个棋子并向左移动，不能越过其他棋子和边界。不能移动者输。问谁有必胜策略。

$n\le 1000, a_i \le 10000$

拓展部分讲义

一、数论问题选讲

1、组合数相关命题：

1）求 $C^1_n, C^2_n, …, C^{n-1}_n$ 的最大公约数；

2）（库默尔定理）求 $V_p (C^m_{m+n})$ ，其中 $p$ 是一个素数。

3）证明：$C^{pb}_{pa} \equiv C^{b}_{a} (mod \ p)$ 对所有素数 $p$，自然数 $a \ge b \ge 0$ 都成立；

2、求 $S(S(S(4444^{4444})))$，其中 $S(x)$ 表示 $x$ 的各位数字之和，如 $S(19) = 1 + 9 = 10$；

3、设 $p$ 为素数，$n$ 为正整数，求证：当且仅当 $p | n - 1$ 时存在正整数 $a, b$ 使得 $2^n = a^p + b^p$ 成立；

4、已知素数集 $M = \{ p_1, p_2, …, p_k \}$，求证：所有满足素因子构成的集合恰好为 $M$ 的正整数的倒数和是一个单位分数 $1 / n$（其中 $n$ 为正整数），且当 $k > 2$ 时，有 $n > 2 \times 3^{k - 2}(k - 2)!$；

5、求证：；

6、证明以下数论定理：

1）$(a^m - 1, a^n - 1) = a^{(m, n)} - 1$（$a > 1, m, n > 0$）；

2）设 $a>b$ 且 $a, b$ 互质，有 $(a^m - b^m, a^n - b^n) = a^{(m, n)} - b^{(m, n)}$；

3）设 $F_n$ 为斐波那契数列的第 $n$ 项，有 $(F_n, F_m) = F_{(m, n)}$；

7、已知 $(a, b) = 1$ ，求：

1）不能用 $ax+by$ 的形式表示的最大正整数，其中 $x, y$ 为非负整数；

2）不能用 $ax+by$ 的形式表示的正整数个数，其中 $x, y$ 为非负整数；

8、威尔逊定理及高斯推论

1）求证：正整数 $p$ 是素数的充要条件为 $(p - 1)! \equiv -1(mod \ p)$；

2）求证：当 $m = 4, p^t, 2p^t$ 时（其中 $p$ 为奇素数，$t$ 为正整数），所有小于 $m$ 且和 $m$ 互素的正整数的乘积模 $m$ 余 $-1$，除此以外该乘积模 $m$ 余 $1$；

9、拓展数论算法

1）解方程 $a^x \equiv b (mod \ p)$；

2）求 $C^m_n (mod \ p)$；

10、二次剩余和勒让德符号

1）（$BZOJ5104$）给定 $a$，求使得 $F_n \equiv a (mod \ 1e9 + 9)$ 成立的最小的 $n$；

2）（$HDU4794$）求斐波那契数列在模 $n$ 意义下的周期。

二、组合数学拓展

1、基本组合恒等式：

1）$k \cdot C^k_n = n \cdot C^{k - 1}_{n - 1}$；

2）$C^{r}_{n} C^m_r = C^m_n C^{r - m}_{n - m}$；

3）$\sum_{i = 0}^{n} (-1)^i C^i_n = 0$；

2、求和

1）$\sum^{n}_{i = 1} i^2 C^i_n$；

2）$C^n_n + C^n_{n + 1} + … + C^n_{n + p}$；

3）$\sum^{n}_{i = 0} (C^i_n)^2$；

3、特殊计数序列：

1）$Catalan$ 数；

2）$Stirling$ 数；

3）整数分拆；

4）格路径计数；

4、求 $1^k + 2 ^ k + … + n^k$。
